💡 Vibe Coding이란?

Vibe Coding은 복잡한 코딩 지식 없이도 자연어로 AI에게 원하는 결과(웹페이지)를 설명하고, AI가 생성한 코드를 실행해보며, 마음에 들지 않으면 다시 피드백을 주어 개선하는 방식이에요.

마치 AI 개발자와 대화하며 함께 홈페이지를 만드는 것과 같다고 생각하면 됩니다!

🎯 Vibe Coding 워크플로우 개요

💡 목표 설정
무엇을 만들지 명확하게 정의하고 AI에게 초기 프롬프트 작성
2
💻 코드 생성
AI가 HTML, CSS 코드를 생성하고 브라우저에서 실행

주석 위치에 실제 정보를 입력하여 웹페이지 완성

1️⃣ 단계 1: 목표 설정 및 초기 프롬프트 작성 💡

이 단계에서는 AI에게 무엇을 만들지 명확하게 알려줍니다.

항목 설명 예시 (음식점/카페)
대상 웹 개발 경험이 없는 소상공인 웹 개발 경험이 전혀 없는 카페 사장님을 위한 홈페이지
목적 Figma 또는 MVP 생성 메뉴 소개, 영업시간, 위치를 보여주는 간단한 MVP 형태의 HTML 웹페이지를 생성
제약사항 Gemini 내의 GEMS 사용 명시 Gemini GEMS를 사용하여 HTML, CSS, JavaScript 코드를 생성해 줘.
핵심 요청 웹페이지의 구성과 목표 반응형으로 디자인된 메인 페이지(index.html)와 메뉴 소개 페이지(menu.html)를 만들어 줘. 디자인은 따뜻하고 깔끔한 느낌으로 부탁해.
추가사항 (자동화) 기본 정보 입력 자동화 방안 상품/서비스 소개, 영업시간, 위치, 연락처 등 기본 정보를 나중에 쉽게 수정할 수 있도록 HTML 코드 내에 주석 처리하여 표시해 줘.

📝 프롬프트 예시

"저는 웹 개발 경험이 없는 소상공인 (카페) 사장님입니다. Gemini GEMS를 사용하여 따뜻하고 깔끔한 느낌의 반응형 웹페이지 MVP를 만들고 싶어요. 메인 페이지 (index.html)는 카페 소개와 핵심 정보를 담고, 메뉴 소개 페이지 (menu.html)는 메뉴 사진과 가격을 보여주도록 HTML과 CSS 코드를 생성해 주세요. 또한, 제가 나중에 상품/서비스 소개, 영업시간, 위치, 연락처 같은 기본 정보를 쉽게 입력할 수 있도록 코드에 한글 주석으로 해당 위치를 명확히 표시해 주세요."

2️⃣ 단계 2: AI (GEMS) 코드 생성 및 실행 💻

AI가 요청한 내용에 따라 HTML, CSS 코드를 생성합니다.

  1. GEMS에서 프롬프트 입력: 단계 1에서 작성한 프롬프트를 GEMS에 입력합니다.
  2. AI 코드 생성: GEMS가 요청에 맞는 HTML 파일명과 해당 파일의 코드를 생성해 줍니다.
    • index.html: 메인 페이지 코드
    • menu.html: 메뉴 소개 페이지 코드
    • style.css (또는 <style> 태그 내부): 디자인을 위한 CSS 코드
  3. 코드 실행 및 확인: 생성된 코드를 복사하여 내 컴퓨터에 .html 파일로 저장하고 웹 브라우저 (크롬, 엣지 등)에서 열어봅니다.
💡 초보자 팁: 웹 개발 경험이 없다면 코드를 저장하고 실행하는 방법을 GEMS에게 물어볼 수도 있어요!
"생성된 index.html과 style.css 파일을 어떻게 저장하고 웹 브라우저에서 열어볼 수 있는지 초보자가 이해하기 쉽게 설명해 줘."

3️⃣ 단계 3: 피드백 및 코드 개선 (반복) 🔄

실행한 결과가 원하는 'Vibe'와 맞는지 확인하고, 마음에 들지 않는 부분을 AI에게 수정 요청합니다. 이 과정이 Vibe Coding의 핵심입니다.

발견된 문제/개선 사항 피드백 (프롬프트 예시)
디자인 "메인 페이지의 배경색이 너무 어둡네요. 밝은 아이보리색으로 바꿔주세요."
구성 "메뉴 페이지에 메뉴 이미지를 4개 넣을 수 있는 공간을 격자(Grid) 형태로 추가해 주세요."
기능 "모든 페이지 상단에 '카카오톡 문의' 버튼을 추가하고, 버튼을 누르면 제 연락처(010-1234-5678)로 연결되게 해주세요."
추가 정보 입력 "index.html의 '영업시간' 주석 아래에 '오전 10시 ~ 오후 8시, 월요일 휴무'를 추가해 주세요."

4️⃣ 단계 4: 완성 및 정보 입력 ✍️

반복적인 피드백을 통해 원하는 형태의 웹페이지 (Figma/MVP 수준)가 완성되면, AI가 주석으로 표시해 둔 곳에 실제 소상공인 정보를 입력합니다.

웹페이지 구성 (예시)

구분 파일명 HTML 주요 내용
메인 페이지 index.html 상단 로고, 카페 소개 (배너), 갤러리, 연락처 및 위치 정보
메뉴 소개 menu.html 커피/음료/디저트 분류, 메뉴 이미지 및 가격 목록

💾 생성된 HTML 파일 확인 방법

AI가 생성한 HTML 코드를 실제로 확인하는 방법을 단계별로 안내합니다.

🪟 Windows 사용자

  1. 메모장 열기: 시작 메뉴에서 '메모장' 검색 후 실행
  2. 코드 붙여넣기: AI가 생성한 HTML 코드를 전체 선택(Ctrl+A) 후 복사(Ctrl+C)하여 메모장에 붙여넣기(Ctrl+V)
  3. 파일 저장:
    • 메모장 상단 메뉴에서 '파일' → '다른 이름으로 저장' 클릭
    • 파일 형식을 '모든 파일'로 변경
    • 파일 이름을 index.html로 입력 (반드시 .html 확장자 포함)
    • 바탕화면이나 원하는 폴더에 저장
  4. 파일 열기: 저장한 파일을 더블클릭하면 기본 웹 브라우저(크롬, 엣지 등)에서 자동으로 열립니다

🍎 Mac 사용자

  1. 텍스트 편집기 열기: Spotlight(⌘+Space)에서 '텍스트 편집' 검색 후 실행
  2. 일반 텍스트로 변경: 상단 메뉴 '포맷' → '일반 텍스트로 전환' 선택
  3. 코드 붙여넣기: AI가 생성한 HTML 코드를 복사(⌘+C)하여 붙여넣기(⌘+V)
  4. 파일 저장:
    • '파일' → '저장' (⌘+S) 클릭
    • 파일 이름을 index.html로 입력
    • 바탕화면이나 원하는 폴더에 저장
  5. 파일 열기: 저장한 파일을 더블클릭하면 Safari나 Chrome에서 자동으로 열립니다
🔍 결과 확인 팁:
  • 파일을 열었는데 코드가 보인다면 → 파일 확장자가 .txt가 아닌 .html인지 확인하세요
  • 여러 HTML 파일(index.html, menu.html 등)을 만들 때는 같은 폴더에 저장하세요
  • CSS 파일(style.css)도 같은 방법으로 저장하되, 확장자를 .css로 저장하세요
  • 수정 후 다시 확인하려면 → 웹 브라우저에서 F5 키(새로고침)를 누르세요

📱 모바일에서 확인하는 방법

⚠️ 권장사항: HTML 파일 생성과 수정은 PC에서 하는 것이 좋습니다. 모바일에서는 완성된 웹페이지를 호스팅한 후 확인하는 것을 추천드립니다.

호스팅 서비스: GitHub Pages, Netlify, Vercel 등을 활용하면 무료로 웹사이트를 공개할 수 있습니다.

⚙️ GEMS를 통한 기본 정보 입력 자동화 방안

GEMS를 사용하여 소상공인이 쉽게 정보를 입력/수정할 수 있도록 하는 가장 좋은 방법은 코드가 아닌 주석을 활용하는 것입니다.

📋 적용 예시 (HTML 코드)

<!-- ✏️ 아래에 카페 소개 문구를 입력하세요 --> <p id="introduction"> "신선한 원두와 따뜻한 공간이 있는 동네 카페, 코지 플레이스입니다." </p> <div class="info-block"> <h3>영업시간</h3> <!-- ✏️ 아래에 영업시간을 입력하세요 --> <p id="hours"> 평일 10:00 ~ 20:00, 주말 11:00 ~ 22:00 (매주 월요일 휴무) </p> </div> <div class="info-block"> <h3>위치 및 연락처</h3> <!-- ✏️ 아래에 주소를 입력하세요 --> <p id="address"> 서울시 강남구 테헤란로 123 </p> <!-- ✏️ 아래에 전화번호를 입력하세요 --> <p id="contact"> 02-1234-5678 </p> </div>
💡 핵심 포인트: AI에게 특정 정보가 들어갈 자리를 한글 주석으로 명확하게 표시해 달라고 요청하면, 소상공인은 코드의 다른 복잡한 부분은 건드리지 않고 주석 바로 아래나 옆에 있는 내용만 수정하여 홈페이지 정보를 쉽게 업데이트할 수 있습니다.
